
# 📊 Human Resource Analytics Dashboard (Tableau Project)

## 🔍 Project Overview

This project focuses on analyzing Human Resource (HR) data to uncover insights related to employee attrition, workforce distribution, and organizational performance. Using Tableau, an interactive dashboard was developed to help HR teams make data-driven decisions and improve employee retention strategies.

---

## 🎯 Objectives

* Analyze employee attrition trends
* Identify key factors influencing employee turnover
* Evaluate department-wise and role-wise distribution
* Study demographic patterns (age, gender, education)
* Provide actionable insights for HR decision-making

---

## 📁 Dataset Description

The dataset contains employee-level information, including:

* Employee ID
* Age
* Gender
* Department
* Job Role
* Monthly Income
* Years at Company
* Job Satisfaction
* Attrition (Yes/No)

---

## 🧹 Data Cleaning & Preparation

* Handled missing and inconsistent values
* Converted categorical fields into usable formats
* Standardized column names and data types
* Created calculated fields for KPI analysis
* Ensured data consistency for visualization

---

## 🧮 Key Metrics & Calculations

* **Attrition Rate (%)**
* **Average Monthly Income**
* **Employee Count**
* **Average Years at Company**
* **Job Satisfaction Score**

---


## 📈 Key Insights

* Higher attrition observed among employees with lower tenure
* Certain departments show significantly higher turnover
* Job satisfaction has a direct impact on employee retention
* Younger employees tend to switch jobs more frequently

---

## 💡 Business Recommendations

* Improve employee engagement and satisfaction programs
* Focus on retention strategies for high-risk departments
* Offer growth opportunities for early-career employees
* Implement data-driven HR policies

---

## 🛠 Tools & Technologies

* Tableau Desktop
* Microsoft Excel / CSV
* Data Visualization Techniques
