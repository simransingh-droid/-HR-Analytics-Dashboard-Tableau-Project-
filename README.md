
📊 HR Analytics Dashboard (Tableau Project)

📌 Overview

This project presents an interactive Human Resource Analytics Dashboard built using Tableau Desktop.
The dashboard provides insights into employee distribution, attrition trends, performance analysis, and satisfaction levels to support data-driven HR decisions.

🎯 Objectives

-Analyze employee attrition and turnover trends.<br>
-Understand workforce distribution across departments.<br>
-Evaluate employee performance and satisfaction.<br>
-Identify key factors affecting employee retention.<br>

🧹 Data Preparation

Cleaned inconsistent values and handled missing data,
Standardized categorical fields (Gender, Department, Attrition),

Created calculated fields for better analysis:

Attrition Rate

Performance Score

Performance Categories


🎛 Interactive Filters

Department

Gender

Age Group

Performance Category

These filters allow dynamic exploration across all dashboards.


🔍 Key Insights

🔄 Attrition rate is relatively high (~33%), indicating retention challenges

🏢 Majority of employees belong to the Production department

📈 Workforce is concentrated in the 35–45 age group

⭐ Most employees fall under the “Fully Meets” performance category

😊 Employee satisfaction is generally high, but does not fully align with retention
